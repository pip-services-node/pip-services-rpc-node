    Calls a remote service's method using the given method, the resolved URI, the set base path, the given route, and the parameters and/or data that was passed.

    throws

    an UnknownException if an unknown method is given.

    Parameters

    • method: string

      the HTTP method to use ("get", "head", "post", "put", "delete").

    • route: string

      the route to the target command (without a base route, for example: "get_random_quote").

    • Optional correlationId: string

      (optional) unique business transaction id to trace calls across components.

    • Default value params: any = {}

      the parameters to pass to the called method.

    • Optional data: any

      (optional) the data to pass to the called method. If a function is passed, it will be called with the result, instead of the callback.

    • Optional callback: function

      (optional) the function to call with the result of the execution (or with an error, if one is raised). If omitted - errors will be thrown instead.
